### 参数
| 参数          | 类型            |   默认值         |   说明   |
|-------------  |---------------- |----------------  |-------- |
| width          | `String`     |    `200px`        |     输入框宽度     |
| format          | `String`     |    `yyyy-MM-dd hh:mm:ss`       |     日期格式   |
| readonly          | `Boolean`     |    `true`        | 是否只读 |
| disabled          | `Boolean`     |    `false`        | 是否可用 |

### 事件

| 名称          |   说明          |      回调参数      |  
|-------------  |---------------- |---------- |
| hide | 隐藏 | (currentValue) |
| change | 改变 |(currentValue) |
