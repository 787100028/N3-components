### 参数
| 参数          | 类型            |   默认值         |   说明   |
|-------------  |---------------- |----------------  |-------- |
| format          | `String`     |    `hh:mm:ss`        |     格式     |
| hour-range          | `String`     |    `[0,60]`      |     小时范围     |
| minute-range          | `String`     |    `[0,60]`      |     分钟范围     |
| second-range          | `String`     |    `[0,60]`       |     秒范围     |

### 事件

| 名称          |   说明          |  参数 |        
|-------------  |---------------- |  ----- |
