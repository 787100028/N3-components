| 参数          | 类型            |   默认值         |   说明   |
|-------------  |---------------- |----------------  |-------- |

| ontext          | `String`     |    `ON`        |     开文案     |
| offtext          | `String`     |    `OFF`        |     关文案     |
| disabled          | `boolean`     |    `false`        |     禁用     |
