### n3-container参数

| 参数          | 类型            |    默认值          | 说明   |
|-------------  |---------------- |----------------  |-------- |
| fluid          | `Boolean`     |    `false `          |    是否流式布局    |


### n3-column参数

| 参数          | 类型            |   默认值         |   说明   |
|-------------  |---------------- |----------------  |-------- |
| col          | `Number`    |    -         |     占的列数     |
| mode          | `String`     |    `md`          |     布局模式 `xs`(手机) `sm`(平板) `md`(桌面) `lg`(大屏幕)    |
| offset          | `Number`     |    -          |     偏移值    |
